# Air Hockey - AI Game

A vibrant, feature-rich 2D Air Hockey game built with Phaser 3, featuring a unique stage-based campaign, a custom shop, and local multiplayer. Created by Antigravity (AI) in collaboration with a very strict (child) project manager.
